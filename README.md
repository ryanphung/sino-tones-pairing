# Getting Started

[Optional] Create a virtual environment:
```
python3 -m venv env
source env/bin/activate
```

Install project requirements:
```
pip install -r requirements.txt
```

Run Jupyterlab:
```
jupyter lab
```

Run the notebook `sino-tones-pairing.ipynb`
